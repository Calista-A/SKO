How This Works:

* Instance Mode: Each sketch (Project1.js and second_page.js) is treated as its own p5 instance, so they won’t interfere with each other. You can use new p5(instanceName) to start a new sketch and stop the previous one.

* Dynamic Switching: When the arrow button is clicked, the current sketch (Project1) is removed, and the next sketch (secondPage) is initialized. This provides a smooth transition between the two pages.

Conclusion:

By using p5's instance mode, you can effectively switch between sketches (pages) in a single HTML file. This avoids conflicts between different setup() and draw() functions and allows dynamic switching between "pages." (Chatgpt, 2024)
